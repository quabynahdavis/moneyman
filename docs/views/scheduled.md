# Scheduled Transactions

Route: `/scheduled`

## Purpose

Manage recurring transactions (bills, subscriptions, periodic income).

## Features

- List of active schedules with frequency, amount, and next date
- Auto vs. Manual badge indicator
- "New Schedule" button (placeholder)
- Empty state when no schedules exist

## Sample Data

Shipped with two demonstration items (Monthly Rent, Netflix Subscription) to illustrate the UI.
