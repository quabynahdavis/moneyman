# Reports

Route: `/reports`

## Purpose

Generate key financial statements. Currently a placeholder with card-based navigation to report types.

## Planned Report Types

| Report | Description |
|---|---|
| Balance Sheet | Assets = Liabilities + Equity snapshot |
| Profit & Loss | Income − Expenses over a date range |
| Cash Flow | Operating + investing + financing activity |
| Portfolio Valuation | Investment account market values |

## Features

- Card grid with icons for each report type
- Click to activate and view report area
- Export button (placeholder for CSV/PDF)
